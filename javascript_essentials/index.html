Lesson 1: Variables, Data Types, and Operators

Variables

Variables are containers for storing data. In JavaScript, we use let, const, or var to declare variables.

// Variable declarations
let name = "JavaScript Learner";
const PI = 3.14159;
var oldWay = "Not recommended anymore";

Data Types

JavaScript supports various data types such as Strings, Numbers, Booleans, Null, Undefined, and Objects.

// Data types examples
let stringExample = "This is a text";
let numberExample = 42;
let booleanExample = true;
let nullExample = null;
let undefinedExample;
let objectExample = { name: "JavaScript", year: 1995 };

Operators

Operators are used to perform operations on values.

// Arithmetic operators
let sum = 5 + 10;
let difference = 20 - 5;
let product = 4 * 3;
let quotient = 20 / 4;

// Comparison operators
let isEqual = 5 === 5;
let isGreater = 10 > 5;

// Logical operators
let andResult = true && false;
let orResult = true || false;

Lesson 2: Functions, Loops, and Conditional Statements

Functions

Functions group reusable code and can use parameters to make them dynamic.

// Function declaration
function greet(name) {
    return `Hello, ${name}!`;
}

// Function expression
const square = function(num) {
    return num * num;
};

Loops

Loops repeat code until a condition is met.

// For loop
let forResult = "";
for (let i = 0; i < 5; i++) {
    forResult += i + " ";
}

// While loop
let j = 0;
let whileResult = "";
while (j < 5) {
    whileResult += j + " ";
    j++;
}

// For...of loop (ES6)
const fruits = ["Apple", "Banana", "Cherry"];
let forOfResult = "";
for (const fruit of fruits) {
    forOfResult += fruit + " ";
}

Conditional Statements

Conditional statements control the flow of the program based on conditions.

// If statement
let age = 20;
let ifResult = "";
if (age >= 18) {
    ifResult = "You are an adult";
} else {
    ifResult = "You are a minor";
}

// Switch statement
let day = 3;
let dayName;
switch (day) {
    case 1:
        dayName = "Monday";
        break;
    case 2:
        dayName = "Tuesday";
        break;
    case 3:
        dayName = "Wednesday";
        break;
    default:
        dayName = "Unknown day";
}

Lesson 3: Arrays and Objects

Arrays

Arrays store multiple values in a single variable.

// Array creation
const colors = ["Red", "Green", "Blue"];

// Accessing array elements
const secondColor = colors[1];

// Array methods
colors.push("Yellow");
const removed = colors.pop();
const joined = colors.join(", ");

Objects

Objects use key-value pairs to group related data.

// Object creation
const person = {
    name: "John Doe",
    age: 30,
    profession: "Developer",
    greet: function() {
        return `Hello, my name is ${this.name}`;
    }
};

// Accessing object properties
const personName = person.name;
const personAge = person["age"];

// Object methods
const greeting = person.greet();

Lesson 4: ES6+ Features

Arrow Functions

Arrow functions provide a shorter syntax for writing functions.

// Regular function
function add(a, b) {
    return a + b;
}

// Arrow function
const addArrow = (a, b) => a + b;

// Arrow function with body
const greetArrow = name => {
    const greeting = `Hello, ${name}!`;
    return greeting;
};

Destructuring

Destructuring makes it easy to extract values from arrays or objects.

// Array destructuring
const coordinates = [10, 20, 30];
const [x, y, z] = coordinates;

// Object destructuring
const user = {
    firstName: "Jane",
    lastName: "Doe",
    email: "jane@example.com"
};
const { firstName, email } = user;

Template Literals

Template literals allow easier string interpolation and multiline strings.

// String concatenation (old way)
const oldWayString = "Hello, " + firstName + "!";

// Template literal
const newWayString = `Hello, ${firstName}!`;

// Multiline string
const multiline = `This is a
multiline string
using template literals`;

Selecting Elements

JavaScript can select HTML elements from the DOM to manipulate them.

// Select elements
const byId = document.getElementById("demo-element");
const byClass = document.getElementsByClassName("example");
const byQuery = document.querySelector(".example");
const byQueryAll = document.querySelectorAll("p");

Modifying Elements

Change content, attributes, and styles of HTML elements.

// Change content
document.getElementById("modify-content").textContent = "Modified content";

// Change attributes
document.getElementById("modify-attr").setAttribute("title", "New title");

// Change styles
const styleElement = document.getElementById("modify-style");
styleElement.style.color = "blue";
styleElement.style.fontSize = "20px";

Lesson 6: Event Handling

Basic Event Handling

Events allow JavaScript to respond to user actions.

// Event listener method
document.getElementById("click-button").addEventListener("click", function() {
    document.getElementById("click-output").textContent = "Button was clicked!";
});

Event Object

The event object contains information about the event that occurred.

// Using the event object
document.getElementById("event-demo").addEventListener("click", function(event) {
    const output = document.getElementById("event-output");
    output.textContent = `Clicked at coordinates: (${event.clientX}, ${event.clientY})`;
    output.textContent += `\nTarget element: ${event.target.tagName}`;
});
Click anywhere in this area

Event Delegation

Event delegation allows handling events for multiple elements with a single listener.

// Event delegation
document.getElementById("parent-container").addEventListener("click", function(event) {
    if (event.target.classList.contains("child-element")) {
        document.getElementById("delegation-output").textContent = 
            `Clicked on ${event.target.textContent}`;
    }
});

Best Practices

Minimize DOM Access

Store references to frequently accessed elements in variables.

// Bad practice (multiple DOM accesses)
function updateCounterBad() {
    document.getElementById("counter-value").textContent = 
        parseInt(document.getElementById("counter-value").textContent) + 1;
}

// Good practice (store reference)
function updateCounterGood() {
    const counterElement = document.getElementById("counter-value");
    counterElement.textContent = parseInt(counterElement.textContent) + 1;
}
Counter: 0

Use Event Delegation

Especially important for dynamic content.

// Good practice - event delegation
document.getElementById("dynamic-list").addEventListener("click", function(event) {
    if (event.target.tagName === "LI") {
        event.target.classList.toggle("selected");
    }
});

// Add new items
document.getElementById("add-item").addEventListener("click", function() {
    const list = document.getElementById("dynamic-list");
    const newItem = document.createElement("li");
    newItem.textContent = `Item ${list.children.length + 1}`;
    list.appendChild(newItem);
});

Avoid Inline JavaScript

Keep JavaScript code separate from HTML for cleaner, more maintainable code.

// Bad practice (inline)
// <button onclick="alert('Hello')">Click Me</button>

// Good practice (separate)
document.getElementById("clean-button").addEventListener("click", function() {
    alert("Hello from clean, separated JavaScript!");
});

Use classList for Class Manipulation

The classList API is cleaner than manipulating the className string.

// Bad practice (manipulating className)
function toggleClassBad() {
    const element = document.getElementById("class-demo");
    if (element.className.includes("highlight")) {
        element.className = element.className.replace("highlight", "");
    } else {
        element.className += " highlight";
    }
}

// Good practice (using classList)
function toggleClassGood() {
    document.getElementById("class-demo").classList.toggle("highlight");
}
